# ChromaDB Vector Database Demo

Simple Vector Database demo using ChromaDB and Python.

---

# Install Dependencies

Install ChromaDB:

```bash
pip install chromadb
```

If you face SQLite compatibility issues on Linux:

```bash
pip install pysqlite3-binary
```
